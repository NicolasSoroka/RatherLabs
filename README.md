
# Getting started ☕


## Installation ⚙

Clone repo

```bash
  git clone https://github.com/NicolasSoroka/RatherLabs.git
  cd RatherLabs
```
Install dependencies and run project
  ```bash
  yarn install
  yarn dev
```
